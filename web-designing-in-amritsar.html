If you have a keen interest in Web Designing, then Webcooks is the right place for you. Along with providing the service of web designing, we also train students in this field. Training will be accomplished based on following web designing technologies.
  • HTML: It stands for Hypertext Markup Language, used to create the structure of web pages. It is used with a combination of CSS and scripting languages.
  • CSS: Cascading Stylesheets are used to give styling to the structure of web pages made with the help of Html.
  • Bootstrap: It is a CSS framework for providing responsive web pages. Along with this, it contains buttons, headers, sidebars, and many other components.
  • Javascript: It’s a client-side scripting language, which is used to handle the behaviour of web pages on the Internet.
  • Jquery: A Javascript library is used to make animated HTML pages and perform event handling.
  • Android Designing: It is done to make the app look well organized and beautiful.
  • Wordpress: It is a Content Management System used with PHP and MySQL.
  • Photoshop: It is a graphics editor used to make amazing graphics.
